# Demo 8: Copy File

Create a file and copy it to create a backup.

## Step 1: Create File

Step [1/1] - Create source file
Action: Create file
File Path: example/output/important.txt

```
Critical Data

This file should be backed up regularly.
```

## Step 2: Copy File

Step [1/1] - Copy file to backup
Action: Copy file
File Path: example/output/important.txt
Destination: example/output/important.backup.txt
